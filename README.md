The source code for my personal portfolio website, available live at **[ramcreddy-ch.github.io](https://ramcreddy-ch.github.io)**.

This portfolio is designed to showcase real-world engineering outcomes in **DevSecOps**, **MLOps**, **Kubernetes Platform Engineering**, and **Real-Time Streaming (Kafka)**.

## Built With

* **HTML5** (Clean, semantic structure)
* **CSS3** (Custom dark theme, responsive grid/flexbox, CSS variables for design tokens)
* **Vanilla JavaScript** (Intersection Observers for scroll animations, custom typing effects)
* **Zero Dependencies** (No heavy frontend frameworks, ensuring maximum performance and instant load times)

## Running Locally

To preview the site locally on your machine, simply run a Python HTTP server in this directory:

```bash
python -m http.server 8090
```
Then navigate to `http://localhost:8090` in your browser.

## Deployment

Pushes to the `main` branch are automatically deployed via GitHub Pages from the root directory.
